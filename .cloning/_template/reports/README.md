# reports/

Verification artifacts produced by QA roles and `npm run compare-screenshots`.

Contents:
- Visual-diff PNGs / overlays comparing clone vs. reference at identical
  viewport / scroll / state.
- Numeric mismatch reports (per section and overall).
- Functional / interaction QA notes and completion reports.

These reports drive the Visual QA/fixer and Functional QA roles. Fixes must be
measured (based on this report), not speculative rewrites.
