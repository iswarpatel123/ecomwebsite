# reference/

Original screenshots of the reference site, captured by `npm run capture-reference`.

Organize by viewport / state, e.g.:
- `1440x900/` (desktop)
- `768x1024/` (tablet)
- `390x844/`  (mobile)

Each image is a ground-truth reference used by `npm run compare-screenshots` and
the Visual QA role to measure fidelity of the clone.
